# Задание 1 — найденные баги по скриншоту


## Найденные дефекты

- **P1 — не работает фильтр по полу:** выбран «Мальчик», но в выдаче есть «девочка».
- **P1 — не работает фильтр по цене:** указан диапазон 1 000–30 000 ₽, но есть объявление за 500 ₽.
- **P1 — не работает фильтр по локации:** при поиске в Москве показываются объявления из Новосибирска и Ижевска.

- **P2 — отображается ошибка при загруженной выдаче:** видно сообщение «Ошибка, обновите страницу».
- **P2 — некорректный фильтр возраста:** в фильтре «Возраст» указано «Щенок» для кошек.
- **P2 — неверная подсказка у иконки избранного:** при наведении на лайк показывается «Не показывать мне похожие объявления».
- **P2 — нерелевантная выдача по породе:** при выборе персидской породы показан экзотический короткошерстный котёнок.
- **P2 — в выдаче есть нерелевантный товар:** показывается «Персидский котёнок плюшевый».
- **P2 — некорректный географический переключатель:** при поиске по Москве отображается «Сначала из Можайска».

- **P3 — опечатка в кнопке:** сверху указано «корьера в Авито».
- **P3 — tooltip перекрывает карточку объявления.**
- **P3 — визуальное наложение элементов в карточке.**

## Комментарий по приоритетам

- `P1` — баги, которые ломают основной сценарий поиска по фильтрам.
- `P2` — заметно ухудшают пользовательский опыт и/или вводят в заблуждение.
- `P3` — локальные визуальные и текстовые дефекты.
